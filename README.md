# 🦁 Shakooti - نظام شكوتي الذكي لإدارة الشكاوى الإنسانية

نظام "شكوتي" هو منصة رقمية متطورة مصممة لدعم المؤسسات الإنسانية والجمعيات في إدارة الشكاوى الميدانية. يتميز النظام بدمج تقنيات الذكاء الاصطناعي (AI) لتحليل النصوص وتقدير مستويات الخطورة تلقائياً لتسريع الاستجابة.

## ✨ المميزات الرئيسية
*   **🧠 تحليل ذكي (AI-Powered):** استخدام نماذج Groq LLM (Llama 3) لتلخيص الشكاوى، تصنيفها قطاعياً، وتقدير درجة خطورتها آلياً.
*   **📱 واجهة ميدانية للباحثين:** نموذج إدخال ذكي يدعم تسجيل بيانات المشتكي، اسم المخيم، ورقم الحالة بدقة.
*   **📊 لوحة تحكم إدارية (Admin Dashboard):** إحصائيات حية، رسوم بيانية للنشاط الأسبوعي، وإدارة الصلاحيات.
*   **🛡️ أمان عالٍ:** نظام توثيق (JWT) مع صلاحيات محكمة تفصل بين مهام الباحث والمدير.
*   **🌙 وضع ليلي (Dark Mode):** واجهة مستخدم عصرية توفر تجربة مريحة في كافة ظروف الإضاءة.
*   **📥 تصدير البيانات:** إمكانية تصدير التقارير إلى ملفات Excel/CSV للتحليل الخارجي.

## 🚀 التقنيات المستخدمة
*   **Front-end:** React.js, Vite, Lucide Icons, Recharts.
*   **Back-end:** Django REST Framework (DRF).
*   **Database:** SQLite (يمكن تحويلها لـ PostgreSQL بسهولة).
*   **AI Engine:** Groq API (Llama-3.3-70b-versatile).
*   **Styling:** Vanilla CSS (Custom Design System).

## 🛠️ التنصيب والتشغيل
1. قم بإنشاء بيئة وهمية (Virtual Environment) وتثبيت المتطلبات:
   ```bash
   pip install -r requirements.txt
   ```
2. قم بتثبيت حزم الـ React:
   ```bash
   npm install
   ```
3. أضف مفتاح الـ API الخاص بـ Groq في ملف `settings.py`.
4. ابدأ السيرفر:
   ```bash
   python manage.py runserver
   npm run dev
   ```
